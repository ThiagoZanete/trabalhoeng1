\doxysubsection{Detailed Description}
Arquivo principal responsável por executar os testes funcionais do simulador. 

Este arquivo contém apenas a função {\ttfamily \doxylink{funcional_2main_8cpp_a840291bc02cba5474a4cb46a9b9566fe}{main()}}, cuja função é chamar os testes funcionais implementados em {\ttfamily functional\+\_\+tests.\+cpp}. Ele serve como ponto de entrada para validar o comportamento do simulador completo, verificando a interação entre \doxylink{classModel}{Model}, \doxylink{classSystem}{System} e \doxylink{classFlow}{Flow} por meio dos cenários definidos nos testes.

A execução inclui\+:
\begin{DoxyItemize}
\item Teste funcional exponencial.
\item Teste funcional logístico.
\item Teste funcional complexo envolvendo múltiplos fluxos.
\end{DoxyItemize}

Cada teste utiliza asserts para verificar se o simulador está produzindo resultados consistentes e matematicamente corretos.
