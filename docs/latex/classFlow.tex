\doxysubsection{Detailed Description}
Classe base abstrata que define um fluxo entre dois Systems. 

Cada \doxylink{classFlow}{Flow} contém ponteiros para um \doxylink{classSystem}{System} de origem e um \doxylink{classSystem}{System} de destino. O \doxylink{classModel}{Model} utiliza objetos \doxylink{classFlow}{Flow} para computar e aplicar transferências entre Systems em cada iteração da simulação.
