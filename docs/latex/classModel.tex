\doxysubsection{Detailed Description}
Classe responsável por gerenciar sistemas, fluxos e a simulação completa. 

O \doxylink{classModel}{Model} armazena dois conjuntos principais\+:
\begin{DoxyItemize}
\item Uma lista de objetos \doxylink{classSystem}{System}, representando entidades com um valor armazenado.
\item Uma lista de objetos \doxylink{classFlow}{Flow}, que representem operações de transferência de valor entre Systems.
\end{DoxyItemize}

A função principal do \doxylink{classModel}{Model} é executar a simulação entre um tempo inicial e final, aplicando todos os fluxos existentes a cada passo.
