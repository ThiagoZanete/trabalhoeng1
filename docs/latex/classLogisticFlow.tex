\doxysubsection{Detailed Description}
Implementação de \doxylink{classFlow}{Flow} para teste\+: crescimento logístico no target. 

Classe de teste que implementa uma taxa dependente do valor do sistema de destino (target) seguindo uma forma logística discreta.

Fórmula usada (no método execute)\+: $ value = 0.01 \times target->getValue() \times \left(1 - \frac{target->getValue()}{70.0}\right) $
